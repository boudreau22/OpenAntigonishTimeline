# Governance Manifest

## Decision-Making Rules

- **Consensus**: We strive for consensus among contributors for major architectural changes.
- **Review**: All pull requests must be reviewed by at least one maintainer before merging.
- **Prioritization**: Tasks are prioritized based on the project backlog and community feedback.

## Data Ownership

- **Public Data**: Project data (timelines, tasks) is considered public information for the Town of Antigonish residents.
- **Privacy**: Personal information (staff emails) is protected and not exposed publicly unless necessary for contact purposes.
- **Supabase**: Data is hosted on Supabase, with access controls managed via Row Level Security (RLS).

## Transparency Commitments

- **Open Source**: The code is open source and available for inspection.
- **Public Backlog**: The project backlog (`BACKLOG.md`) and roadmap (`ROADMAP.md`) are public.
- **Feedback**: We welcome feedback from the community regarding project features and data accuracy.
